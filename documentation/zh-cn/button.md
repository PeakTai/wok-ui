# 按钮

按钮是页面最常用的点按交互元素，对于一个网页来说绝大多数操作都是通过点按来进行的，所以组件库也提供了
丰富的按钮样式风格。

## 使用示例

通过构造 Button 类型实例来创建按钮。

```ts
new Button({
  // 按钮文字
  text: '开始吧！',
  // 类型，主题色按钮
  type: 'primary',
  // 描边样式，默认为 false ，表示填充风格
  outline: true,
  // 尺寸
  size: 'lg',
  // 处理点击事件
  onClick(ev) {
    // TODO 交互操作
  }
})
```

按钮也被很多其它的内置组件所广泛使用，如模态框等。
