# WOK-UI English Documentation

## Documentation Conventions

All code examples in this documentation are written in TypeScript. It is recommended to use wok-ui only in TypeScript projects, as many features depend on TypeScript's type constraints and inference for optimal development efficiency.

The documentation does not cover every API parameter — only common usage patterns to help readers understand how things work. For detailed API information, refer to the TypeScript definition files. During development, you can also navigate to definition files via your IDE. Most parameters of built-in modules have detailed comments.
